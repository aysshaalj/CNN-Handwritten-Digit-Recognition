# CNN-Handwritten-Digit-Recognition
Using CNN to classify Handwritten Digits

This project builds a Convolutional Neural Network (CNN) to recognize handwritten digits using the MNIST dataset. The model learns to classify grayscale images of digits from 0 to 9.

Dataset
The project uses the MNIST dataset which contains:
- 60,000 training images
- 10,000 testing images
- Image size: 28x28 pixels
- 10 classes (digits 0–9)

Tools Used
- Python
- TensorFlow / Keras
- NumPy
- Matplotlib
- Scikit-learn

Project Steps
1. Load the MNIST dataset
2. Visualize sample digit images
3. Normalize and reshape the data
4. Build a CNN model
5. Train the model
6. Evaluate performance using accuracy and confusion matrix

How to Run
1. Open the notebook in Google Colab or Jupyter Notebook
2. Install required libraries if needed:

pip install tensorflow numpy matplotlib scikit-learn

3. Run all cells in the notebook to train and evaluate the model.
